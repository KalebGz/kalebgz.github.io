Sleep Safe

Sleep Safe is a product designed to actively measure the noise near spaces where someone is sleeping. It is a device that can be set to a predefined threshold for the noise level that would be considered tolerable. Should the noise level exceed this threshold, Sleep Safe sends Bluetooth signals to a node device placed in an area near the source of the noise which illuminates an LED fixture to indicate a warning for noise exceeding the defined threshold. The device also has the functionality to allow the user to override the algorithm and manually trigger the noise level indicator.
